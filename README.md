**Project**
This project is a little blog using [JSONPlaceholder](https://jsonplaceholder.typicode.com/).

**Installation**
You need to follow some steps to run this app:

1.  clone the project
2.  cp .env.example .env and fill in the variables
3.  yarn install
4.  yarn start
